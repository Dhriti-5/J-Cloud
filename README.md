# J-Cloud Distributed File Storage System

J-Cloud is a distributed object storage system inspired by HDFS and AWS S3.

## Features

- Distributed file storage
- File chunking
- Multi-node architecture
- Fault tolerance
- Replication
- Web-based interface

## Architecture

Master Node
Data Nodes
Web Server

## Tech Stack

Java
Servlets
JSP
Sockets
MySQL
Apache Tomcat

## How to Run

1 Start Master Node
2 Start Data Nodes
3 Start Web Application
4 Access via browser
